Crypto Sentiment Score

About this Indicator:

Average sentiment of headlines from cryptocurrency news outlets using finBERT, a large language model for extracting information from financial text

  • Twitter
  • Github

Approach

  1. Scrape: The top 5 crypto-only news sites are scraped using the BeautifulSoup library in Python, capturing the headlines from their home pages.
    • Websites: CoinDesk, Decrypt, Blockworks, BeInCrypto, U.Today
    • File with extracted headlines: link to file on GitHub
  2. Sentiment Analysis: Headlines are analyzed using the finBERT model provided by Hugging Face.
    • File with added sentiments: link to file on GitHub
  3. Store Results: The average sentiment is calculated and stored in a CSV file in a public repository that can be accessed by this wepage.
    • Calculates the average sentiment per day with the following categories Positive (+1), Negative (-1), or Neutral (0).
    • File with calculated averages: link to file on GitHub
    • A GitHub Actions workflow is configured to execute steps 1 through 3 daily at 00:00:00 UTC.
  4. Display on Dashboard: This webpage retrieves the stored scores with JavaScript, fetches the daily Bitcoin prices using coinGecko free API, and updates the graph above.
